Ejemplo de Integración del Chat Widget

Instrucciones

Este es un ejemplo de cómo integrar el chat widget en tu web.

El widget aparecerá automáticamente en la esquina inferior derecha.

Haz clic en el botón flotante para abrir el chat.

Código de Integración

Para usar este widget en tu web, añade este código antes del cierre de </body>:

<!-- Chat Widget -->
<script src="https://tu-dominio.vercel.app/chat-widget.js"></script>
<script>
  ChatWidget.init({
    apiUrl: 'https://tu-dominio.vercel.app/api',
    position: 'bottom-right',
    buttonColor: '#2563eb',
    theme: 'light'
  });
</script>
